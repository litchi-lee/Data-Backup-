李天、徐紫菡、张子彦的2023UESTC软工实验项目--**数据备份软件**

# 项目完成功能：<br>
## 基本要求：  
数据备份、数据还原
## 扩展要求：  
文件类型支持
元数据支持  
打包解包（tar） 
压缩解压（哈夫曼算法） 
加密解密（MD5） 
图形界面（使用Qt设计）
